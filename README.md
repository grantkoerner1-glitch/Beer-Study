# Beer Study Durham Cozy Redesign

Atmosphere-focused single-page website concept for Beer Study Durham with warm visuals, immersive photography, and clear visit information.

## File Structure

- `index.html` — Semantic section layout and content.
- `styles.css` — Cozy visual theme, responsive layout, and transitions.
- `script.js` — Tap list filtering, mobile nav behavior, and reveal animations.

## Run Locally

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`.
