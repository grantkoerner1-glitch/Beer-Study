# Beer Study Durham Website Refresh

Professional-quality two-page site concept for Beer Study Durham with a warm visual identity, cleaner information architecture, and a dedicated full menu experience.

## File Structure

- `index.html` — Homepage with sticky navigation, about, drinks preview, events, and visit details.
- `menu.html` — Dedicated menu page with The Boot Room food menu and Beer Study drinks categories.
- `styles.css` — Shared visual system, responsive layouts, accessibility/focus states, and motion preferences.
- `script.js` — Navigation behavior, smooth scrolling, tap filter rendering, menu data loading, and reveal interactions.
- `menu-data.json` — Structured Boot Room menu source data.

## Run Locally

```bash
python3 -m http.server 8000
```

Open:
- `http://localhost:8000/index.html`
- `http://localhost:8000/menu.html`
